> All Payment Website Milik **TGRS**

Website ini adalah halaman pembayaran resmi milik **TGRS** yang
digunakan untuk menampilkan berbagai metode pembayaran dalam satu
tampilan yang rapi, modern, dan mudah digunakan.

Website ini tidak ditujukan untuk didownload, dikloning, atau
dimodifikasi oleh pihak luar. Halaman ini dibuat khusus sebagai landing
page payment resmi.

---

## 📌 Fungsi Website

TGRS All Payment berfungsi sebagai:

-   Pusat informasi metode pembayaran
-   Landing page transaksi
-   Media branding TGRS
-   Halaman pembayaran yang praktis dan cepat diakses

Pengguna cukup membuka website dan memilih metode pembayaran yang
tersedia.

---

## ✨ Fitur Website

### 🔹 Welcome Screen

Halaman pembuka dengan animasi dan tombol masuk sebelum mengakses daftar
pembayaran.

### 🔹 Background Music

Musik otomatis berjalan setelah masuk ke halaman utama dan dapat
dinonaktifkan melalui tombol toggle.

### 🔹 Collapsible Payment System

Setiap metode pembayaran dapat dibuka dan ditutup secara interaktif
untuk menjaga tampilan tetap bersih.

Metode yang tersedia: - Bank BCA - DANA - GOPAY - OVO - ShopeePay

### 🔹 Copy Nomor Otomatis

Tombol salin untuk menyalin nomor pembayaran secara instan ke clipboard.

### 🔹 Desain Modern

-   Tema merah gelap dengan efek glow
-   Animasi halus
-   Tampilan responsif untuk perangkat mobile
